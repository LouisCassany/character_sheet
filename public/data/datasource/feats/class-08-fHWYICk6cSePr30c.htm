You've learned that it pays for a Pathfinder to be able to cast spells without being noticed. You gain the @UUID[Compendium.pf2e.feats-srd.Item.sIeuPW0j39fTZm08]{Conceal Spell} wizard class feat, though it's not a wizard feat for you.
