Name: Rough Terrain Stance Nom: Posture du terrain accidenté État: libre -- Desc (en) --

Requirements You gained a terrain attunement benefit this turn.

You enter a stance that makes it difficult to move around you. Each square adjacent to you becomes difficult terrain that matches the terrain attunement you gained (rime forms in an arctic terrain, momentary undergrowth shoots up in a forest terrain, and so on). You ignore this difficult terrain.

This stance ends if you move into a different type of terrain.

-- Desc (fr) --

Conditions Vous avez obtenu un avantage de l'harmonisation de terrain au cours de ce tour.

Vous adoptez une posture qui rend difficile de se déplacer autour de vous. Chaque case qui vous est adjacente devient un terrain difficile qui correspond à l'harmonisation du terrain que vous avez obtenu (du verglas se forme sur un terrain arctique, un sous-bois momentané surgit dans un terrain forestier, etc). Vous ignorez ce terrain difficile.

Cette posture se termine si vous vous déplacez sur un type différent de terrain.
