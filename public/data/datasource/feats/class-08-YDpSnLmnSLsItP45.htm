Name: Improvised Critical Nom: Critique improvisé État: libre -- Desc (en) --

You can apply critical specialization effects to improvised weapons you wield. For any given improvised weapon, the GM determines which critical specialization effect is most applicable, based on the type of object it is.

-- Desc (fr) --

Vous pouvez appliquer les effets critiques spécialisés en cas de coup critique avec une arme improvisée que vous maniez. Pour chaque arme improvisée donnée, le MJ détermine l'effet critique spécialisé qui est le plus approprié en fonction du type d'objet.
