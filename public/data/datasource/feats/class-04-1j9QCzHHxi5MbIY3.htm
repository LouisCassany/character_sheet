Name: Big Game Trapper Nom: Piégeur de gros gibier État: libre -- Desc (en) --

You can set snares specifically for big prey, leaving smaller creatures unscathed. Each time you set a snare, you can choose Small, Medium, Large, or Huge. That snare can be triggered only by creatures of that size or larger.

-- Desc (fr) --

Vous pouvez poser des pièges artisanaux spécialement conçus pour le gros gibier, en laissant indemnes les plus petites créatures. Chaque fois que vous posez un piège artisanal, vous pouvez choisir Petite, Moyenne, Grande ou Très grande. Ce piège artisanal ne peut être déclenché que par des créatures de cette taille ou de taille supérieure.
