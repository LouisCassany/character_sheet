Name: Engrave Memory Nom: Mémoire gravée État: libre -- Desc (en) --

You store memories inside a stone. This memory can convey up to 10 minutes of material. When you Cast this Spell, choose a command word. Doing so creates a carved symbol on the stone that hints at the word. Any creature can access the memories stored within the stone by using an action to speak the command word.

As it imparts the memories, the stone crumbles to dust, but the memory is infallibly available to the creature that activated the stone for the next 24 hours. After that time, the creature's normal capacity for memory must be relied upon to recall the stone's imparted memory. It's up to the GM what benefit the imparted memory grants, from circumstance bonuses to checks related to the memory or clear information on subjects the memory covers.

-- Desc (fr) --

Vous stockez des souvenirs à l'intérieur d'une pierre. Cette mémoire peut contenir jusqu'à 10 minutes de matériel. Lorsque vous Lancez ce sort, choisissez un mot de commande. Cela crée un symbole sculpté sur la pierre qui fait allusion au mot. Toute créature peut accéder aux souvenirs stockés dans la pierre en utilisant une action pour prononcer le mot de commande.

Lorsqu'elle transmet les souvenirs, la pierre tombe en poussière, mais la mémoire est infailliblement disponible pour la créature qui a activé la pierre pendant les 24 heures suivantes. Passé ce délai, la créature doit faire appel à ses capacités normales de mémorisation pour se rappeler les souvenirs transmis par la pierre. C'est au MJ de décider quels avantages la mémoire impartie confère, qu'il s'agisse de bonus de circonstances aux tests liés à la mémoire ou d'informations claires sur les sujets couverts par la mémoire.
